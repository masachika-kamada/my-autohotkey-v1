# my-autohotkey

1. Install AutoHotKey **v1.1** from [here](https://www.autohotkey.com/)
2. Clone this repo
3. Create a shortcut for main.ahk
4. Open the startup folder by typing "shell:startup" in the Run dialog (Windows+R)
5. Paste the shortcut created in step 3 into the startup folder that opened in step 4

## Migrating to AutoHotkey v2.0

If you're interested in using AutoHotkey v2.0, you can check out the [
my-autohotkey-v2](https://github.com/masachika-kamada/my-autohotkey-v2) for a version of the script that's compatible with v2.0.

To convert your v1.1 script to v2.0, you can use the [
AHK-v2-script-converter](https://github.com/mmikeww/AHK-v2-script-converter).
